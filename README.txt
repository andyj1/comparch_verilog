How to test PaperProcessor:

1. Change Directory to Paperprocessor file
2. To compile, type iverilog -o testbench testbench.v in terminal, the compiled file is saved as testbench
3. Type ./testbench to run the code. The output should show time, program counter, instruction, output register value and status register value

*You can also change the code in instruct.bin
